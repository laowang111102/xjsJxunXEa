# 旅游数据分析推荐系统 java1090

## 项目概述

本系统基于Python Django框架，致力于通过数据分析为用户提供旅游景点的推荐。通过爬虫采集旅游网站数据，存储于MySQL数据库中，进行多维度统计分析，并通过可视化图表展示。

## 技术栈

- **后端**: Python Django
- **数据库**: MySQL
- **前端**: HTML/CSS/JavaScript（可选）

## 功能模块

### 基础功能

- 用户登录与注册
- 数据展示
- 用户评论
- 用户修改个人信息

### 高级功能

- 算法推荐
- 数据可视化图表
- 管理后台

## 系统角色

- **普通用户**: 登录、浏览数据、发表评论、修改个人信息
- **管理员**: 通过django-admin管理后台进行数据维护

## 运行环境

- Python环境
- Django框架
- MySQL数据库

## 部署步骤

1. 搭建Python与Django环境
2. 配置MySQL数据库
3. 运行数据爬虫，填充数据库
4. 启动Django应用，运行服务器
